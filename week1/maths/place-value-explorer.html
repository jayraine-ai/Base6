What is Place Value?

📍 Where a digit lives matters

In any number, each digit has a place. That place tells us how much that digit is worth. The same digit can mean very different things depending on where it sits.

For example — the digit 3 means something very different in these two numbers:

3,000 — the 3 is worth three thousand
300 — the 3 is worth three hundred

🏠 The four place value columns

Every number up to 10,000 fits into four columns. From left to right:

📊 Order Numbers

How do we put numbers in order?

To order numbers, look at the highest place value column first — the thousands. If those are the same, look at the hundreds. If those are the same, look at the tens, and so on.

Start left and work right — the biggest column decides first.

➕ Calculate

Adding and subtracting with place value

When we add or subtract, we work column by column — ones with ones, tens with tens, hundreds with hundreds. Place value keeps everything lined up correctly.
