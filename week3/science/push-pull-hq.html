⚖️ Push & Pull HQ

Forces are pushing and pulling on everything, all the time — so why isn't everything flying about? Because forces can balance. Run the tug-of-war lab, then read the force arrows like a scientist.

The big idea

When the forces on an object are balanced — equal in size, opposite in direction — the object's motion doesn't change. If it was still, it stays still.

When forces are unbalanced, there's a winner — and the object starts moving, speeds up, slows down or changes direction towards the bigger force. Unbalanced forces are what change motion.
